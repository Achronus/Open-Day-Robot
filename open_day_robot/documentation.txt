Required pre-requisites: ROS installed with gazebo simulation for tiago robot, OpenCV person detection

1) change directory to tiago workspace and source setup files

2) run gazebo with navigation launch file and localize the robot as specified at http://wiki.ros.org/Robots/TIAGo/Tutorials/Navigation/Localization

3) a) patrol mode 
	in a seperate terminal: rosrun open_day_robot patrol
   b) approach mode
	in a seperate terminal: roslaunch pal_person_detector_opencv detector.launch image:=/xtion/rgb/image_raw
	in a seperate terminal: rosrun open_day_robot persondetection
